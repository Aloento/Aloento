### B

Q, Q ⇒ P ∨ R, R ⇒ S ⊢ P ∨ S

```coq
Require Import Classical.

Parameters P Q R S: Prop.

Hypothesis P1: Q.
Hypothesis P2: Q -> P \/ R.
Hypothesis P3: R -> S.
Goal P \/ S.

```

### C

S ⇒ P, Q ⇒ E ⊢ (P ⇒ Q) ⇒ (S ⇒ E)

```coq
Require Import Classical.

Parameters P Q E S: Prop.

Hypothesis P1: S -> P.
Hypothesis P2: Q -> E.
Goal (P -> Q) -> (S -> E).

Proof.

intros H.
assert(S -> E).
intros H1.
apply P2.
apply H.
apply P1.
exact H1.
exact H0.

Qed.
```
