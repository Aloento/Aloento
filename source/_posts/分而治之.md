# Inversion Count

## 定义

给定一个数组 $A$，如果 $i < j$ 且 $A[i] > A[j]$，则称 $(i, j)$ 是 $A$ 的一个逆序对

也就是说，某个前面的元素比后面的元素大，这样的对数就是逆序对

如有 `arr[] = [8, 4 ,2, 1]`  
则逆序对为 `(8, 4), (8, 2), (8, 1), (4, 2), (4, 1), (2, 1)`  
共 6 对

## 简单解法

1. 遍历数组
2. 对于元素 $A[i]$，遍历 $A[i+1:]$，统计比 $A[i]$ 小的元素个数
3. 累加

```ts
const arr = [8, 4, 2, 1];
let count = 0;

for (let i = 0; i < arr.length; i++) {
  for (let j = i + 1; j < arr.length; j++) {
    if (arr[i] > arr[j]) {
      count++;
    }
  }
}

console.log(count); // 6
```

$O(n^2)$
